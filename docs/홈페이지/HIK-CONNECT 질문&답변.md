# 🛠️ 문의 답변

??? success "계정 생성 시 국가가 잘못 선택된 경우 변경 방법"
    **문의:** 계정 생성 시 국가가 잘못 선택된 경우 변경 방법 문의<br>
    **답변:** 계정 생성 시 국가가 잘못되어 문제가 발생한 경우 고객 스스로 국가 변경이 되지 않으며, 계정 영구 삭제 후 재등록이 필요합니다.

??? success "어플 계정의 비밀번호 변경을 위해 보안코드 핸드폰 번호로 요청시 문자가 오지 않는 문제"
    **문의:** DS-K1T342EFWX 모델의 HTTPS 포트 변경 가능 여부<br>
    **답변:** 해당 문제는 두가지 동시 점검이 필요<br>
                 1. 고객 통신사에서 국외발신 문자 차단 여부 확인<br>
                 2. 본사 서버에서 문자가 정상적으로 발송됐는지 확인<br>

??? success "어플에서 아큐센스 기능(사람/차량 구분)이 될 때가 있고 안 될 때가 있음"
    **답변:** 너무 빠르게 어플을 조작할 경우 이벤트 이력을 불러오는 도중에 필터링 명령이 중복으로 들어가서 증상이 발생할 수 있습니다.
