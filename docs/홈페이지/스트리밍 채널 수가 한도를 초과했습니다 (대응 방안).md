\# 스트리밍 채널 수가 한도를 초과했습니다 (대응 방안)



✅스트리밍 채널 수가 한도 초과라는 에러는 녹화기의 스펙중  remote connection 스펙을 초과하는 경우 발생합니다.<br>

Remote connection은 녹화기가 동시에 제공할 수있는 영상 채널수를 의미하고 ivms-4200, HC(핸드폰 어플) 등에서 영상을 동시에 보시는 경우<br>

해당 스펙을 초과하게되어 발생 할 수 있습니다.<br>

 

✅따라서 아래와 같이 확인해 주시기 바랍니다.<br>

 

먼저 현재 ivms-4200, HC어플 등 동시 사용자가 몇명인지 확인(HC동일 계정으로 로그인하여 동시 사용하는 사람 포함)<br>

동시사용자가 많다면 동시사용자를 줄임<br>

동시사용자가 적다면 녹화기가 현재 영상을 전달하고 있는 상태에 갱신을 못하여 발생할 수도 있는 관계로 녹화기 리부팅<br>
