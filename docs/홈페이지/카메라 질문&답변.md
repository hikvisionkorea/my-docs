# 🛠️ 문의 답변

??? success "CVBS에서 NTSC/PAL 변환 방법"
    **답변:** TVI 모드에서 (예: 2MP 25FPS(PAL) / 30FPS(NTSC))와 같이 프레임 선택 후 CVBS 모드로 변경하면 적용됩니다.

??? success "영상 중 A1 깜빡이는 증상 끄는 방법"
    **답변:** 알람 입력 A1 측에 알람이 트리거될 때 발생합니다.<br>메인메뉴 -> 돔세팅 -> 알람`에서 알람 입력을 끄면 해결됩니다.

??? success "Smart Supplement Light 항목의 역할"
    **답변:** 보조등이 켜져 있을 때 이미지가 과도하게 노출(화이트아웃)되는 경우, '스마트 보조 조명 ON'을 선택하여 방지할 수 있습니다.

??? success "플리커(조명 깜빡임) 심한 경우 해결 방법"
    **답변:** `설정 -> 이미지 -> 노출설정 -> 안티밴딩(Anti-Banding)` 기능을 켜주세요.

??? success "영상이 이상하거나 아래 부분만 모자이크처럼 깨지는 증상"
    **답변:** 트래픽 문제일 가능성이 높습니다. <br>비트레이트를 낮추거나 네트워크 망이 단독인지 트래픽 확인이 필요합니다.
