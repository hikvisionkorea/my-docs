\# 📼 SADP 활용 비밀번호 초기화

✅**메일로 발송된 파일은 24시간 이내에만 사용 가능합니다.**<br>
비밀번호 초기화 사이트를 통해 정상적으로 접수가 되었다면 평균 10분 이내 늦어도 30분 이내에 답변을 받으실 수 있습니다. 업무 외 시간에 보내 주신 경우라면 다음 업무일 오전 9시 30분 이전에 받으 실 수 있으며 답변 받으신 파일의 사용 기한은 24시간이 됩니다.  

<br>

✅**비밀번호 초기화 시 오류가 발생하는 경우**<br>
1. PC와 장비 재부팅<br>
2. PC와 장비를 1:1로 랜선으로 연결<br>
3. 3.0.0.10 이상의 SADP를 관리자 권한으로 실행<br>
4. Forgot password에서 Export 후 새로 생성된 파일로 비밀번호 초기화 재접수<br>

순으로 재진행하셔야 합니다.<br>

<br>

비밀번호 초기화 후 펌웨어 업그레이드를 권장드리며<br>
비밀번호 재설정 후 비밀번호 분실 시 직접 초기화가 가능한 UGUID 파일 백업 및 비밀번호 질문 등록을 해주시기 바랍니다.
